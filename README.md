# Hangman
Hangman Game
Welcome to Hangman! This is a simple implementation of the classic word-guessing game built using HTML, CSS, and JavaScript.


How to Play
Open the Game: Open the index.html file in a web browser to start the game.
Guess the Word: Click on the letters displayed on the screen to guess the word.
Win or Lose: Keep guessing until you either guess the word correctly or the full gallows is drawn.
Enjoy: Have fun and challenge yourself with this word-guessing game!
Features
Intuitive Interface: Simple and easy-to-use interface for an enjoyable gaming experience.
Random Words: Endless gameplay with randomly generated words for each round.
Visual Representation: Watch as the gallows and the hangman are drawn with each incorrect guess.
Technologies Used
HTML: Used for creating the structure of the game.
CSS: Used for styling the game's appearance.
JavaScript: Used for implementing game logic and interactivity.
How to Run
To run the game, simply open the index.html file in a web browser.
